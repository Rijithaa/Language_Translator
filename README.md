# 🌍 Language Translator

A simple Language Translation Tool that allows users to translate text between multiple languages using a Translation API. The project provides a clean and responsive interface with features like language selection, text translation, copy-to-clipboard, and text-to-speech.

## 🚀 Features
- Multi-language translation
- Source and target language selection
- Real-time translation
- Copy translated text
- Text-to-Speech support
- Responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Translation API

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Run `index.html` using Live Server.
4. Start translating!

## 📚 Learning Outcomes
- API Integration
- JavaScript Fetch API
- DOM Manipulation
- Frontend Development
